# SmritiLok - Mystic Oasis

A web app to create, store, and share memories, featuring private and public spaces, AI avatars, and interactive tools.

## Features

- Split-screen UI for Private and Public Memories
- Private tools: Chat, Calendar Journal, Files, Maps
- Public tools: AI Avatar, Shared Memories, Memory Wall, Biography
- Responsive design with modern UI components
- Social media integration

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- [pnpm](https://pnpm.io/) (recommended package manager)

### Installation

1. Clone the repository:
   ```sh
   git clone <your-repo-url>
   cd mystic-oasis
   ```

2. Install dependencies:
   ```sh
   pnpm install
   ```

### Development

Start the development server:
```sh
pnpm dev
```
Open [http://localhost:8080](http://localhost:8080) in your browser.

### Build

To build for production:
```sh
pnpm build
```

To start the production server:
```sh
pnpm start
```

## Project Structure

```
client/
  pages/Index.tsx      # Main UI page
  assets/              # Images and static assets
components/ui/         # Reusable UI components
...
```

## Customization

- Update images in `client/assets/`
- Edit UI and logic in `client/pages/Index.tsx`
- Style using [Tailwind CSS](https://tailwindcss.com/)
